# Retail Sales Dataset

This repository contains a retail sales dataset and accompanying Jupyter notebooks for data cleaning and exploratory data analysis (EDA).

- **Dataset**
    - `retail_store_sales.csv` – Original raw dataset (dirty data).  
    - `retail_data_cleaned.csv` – Cleaned dataset produced after processing in the cleaning notebook.
 
- **Notebooks**
    - `Dirty_Retail_Sales.ipynb` – Notebook used to clean and preprocess the raw dataset.  
    - `Retail_Sales_EDA.ipynb` – Notebook used to perform exploratory data analysis on the cleaned dataset.
